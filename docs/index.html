Règles du Jeu

Le joueur qui a les pierres noires commence.
Chacun à leur tour, les joueurs pose une pierre de leur couleur.
Il est impossible de recréer une situation ayant déja exister sur le plateau.
Le jeu de Go se joue sur un plateau de 19x19, mais il peut aussi se jouer sur des plateaux plus petits comme 13x13 ou 9x9.
